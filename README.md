# vite-vue-rack
Element Plus plugin for `Vite` .

## Install

```bash
npx degit guoweiTang/vite-vue-rack my-rack-app
cd my-rack-app
pnpm i
pnpm dev
```

## Project setup

### Compiles and hot-reloads for development and mock data
```
pnpm dev
```
### Compiles and minifies for production
```
pnpm build
```
