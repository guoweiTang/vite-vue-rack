# vue-cli-plugin-rack
Element Plus plugin for `Vite` .

## Install

```bash
git clone git@github.com:guoweiTang/vite-vue-project.git my-app
cd my-app
npm i
npm run dev
```

## Project setup

### Compiles and hot-reloads for development and mock data
```
npm run dev
```
### Compiles and minifies for production
```
npm run build
```
